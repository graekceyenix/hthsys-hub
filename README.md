# hthsys-hub

## 项目简介

hthsys-hub 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何单个域名或具体网站，而是作为通用存放空间，统一管理各类静态页面资源。

## 目录说明

```
/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 页面所需的静态资源（图片、样式、脚本等）
└── README.md       # 本文件
```

- `pages/` 目录下按需组织子目录或直接放置 HTML 文件。
- `assets/` 目录用于存放页面引用的外部资源文件，避免资源散落。

## 页面归档说明

- 每个 HTML 页面应保持独立，不依赖本仓库以外的外部资源（除公共 CDN 外）。
- 页面文件名应清晰标识其内容或用途，建议使用英文小写字母和连字符。
- 如需在页面间共享样式或脚本，可统一放置于 `assets/` 目录下，并在页面中通过相对路径引用。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面时请遵循上述目录结构。
- 不建议在本仓库中存放动态内容或需要后端支持的页面。
- 如果页面引用外部资源，请确保其来源稳定可靠，避免因外部链接失效导致页面异常。
- 欢迎通过 Issue 或 Pull Request 提出改进建议或提交新的归档页面。
